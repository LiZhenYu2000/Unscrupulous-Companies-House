## 贡献信息

### 要求

提供的信息中需要包含准确、真实可查的公司名，同时需要写明不推荐该公司的原因理由，原因理由中的不雅用语或敏感内容会被审核后屏蔽。

您可以选择在提交信息中写明具体的部门或者事业群，使提供的信息更具体且具有真实性，如果担心被公司进行定位，也可以不提供具体的部门。
